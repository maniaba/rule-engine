# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- **Rule Engine Core**: Complete rule evaluation and execution system
- **Flexible Configuration**: Support for array and JSON rule definitions
- **Condition System**: 15+ condition types including comparisons, string operations, array operations, and logical operators
- **Action System**: Extensible action framework with custom callable support
- **Context Management**: ArrayContext for rule data management
- **Builders**: ArrayBuilder and JsonBuilder for rule configuration
- **Validation**: Built-in rule structure validation
- **Evaluator**: BasicEvaluator for rule processing and execution
